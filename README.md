# acinetobacter_phage_analysis

# Project overview
Analyzed 50 publicly available Acinetobacter bacteriophage genomes from Virus-Host DB using Python, Biopython, and Pandas. Calculated genome size and GC content, generated summary statistics and visualizations, and applied Linux-based bioinformatics workflows for genomic data analysis.

# Objectives
- Calculate genome size of each phage
- Calculate GC content
- Visualize genome characteristics
- Explore genomic diversity among Acinetobacter phages

# Dataset
- 50 Acinetobacter phage genome sequences downloaded from Virus-Host DB
- FASTA format

# Methods
1. Download phage genome sequences from NCBI.
2. Parse FASTA files using Biopython.
3. Calculate:
Genome length (bp)
GC content (%)
4. Store results in a pandas DataFrame.
5. Generate summary statistics and visualizations.

# Tools Used
Python 3
Biopython
Pandas
Matplotlib

# Results
The analysis generated:
- Genome length statistics
- GC content statistics
- Histograms and data visualizations
